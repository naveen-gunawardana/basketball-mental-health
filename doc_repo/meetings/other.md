## App Ideas

### 1. Pressure Reset
**Problem:** Under pressure you speed up, panic, lose handle, make late decisions.

**Solution:** 30–60s reset + "calm reps" during practice.

**Features:**
- **20-second check-in:** anxiety/energy/confidence (1–10)
- **"Panic button" routine:**
  - Breathing timer (box or 4-7-8)
  - Cue words (e.g., "LOW + EYES + PACE")
  - 1 tiny action plan (retreat dribble → middle → outlet)
- **Exposure ladder workouts:**
  - Level 1: Timed dribble
  - Level 2: Partner pressure
  - Level 3: Full court 1v1
  - Level 4: Live run constraints

**Metrics:**
- Pre/post anxiety score
- Turnovers per run
- "Escape success rate" (get across half without pickup?)

---

### 2. Hoops Progress Tracker ⭐ (Simplest MVP)
**Problem:** Skill plateaus + motivation dying when progress feels invisible.

**MVP:** Weekly micro-tests (FT%, 5-spot makes, handle control) → graphs + streaks + "next focus" suggestion.

**Why it's strong:** Easiest to build, easiest to get users, easy to prove impact (before/after stats).

---

### 3. Plateau Breaker (14-Day Plan Generator)
**Problem:** "I'm grinding but not improving."

**MVP:** User picks role (3&D, slasher, big) + weakness → app outputs 15–25 min/day plan for 14 days, adaptive difficulty.

**Extra:** Day 1 baseline test → Day 14 retest.

---

### 4. Coach Conversation Script Builder
**Problem:** Players don't know how to talk to coach / fear retaliation.

**MVP:** Input situation → generates 3 respectful questions + 1-page "here's what I'm working on" note.

**Add-on:** Post-game reflection → 2 controllables for next week.

---

### 5. Sleep + Homework + Practice Planner
**Problem:** Late practices + homework → chronic sleep loss → worse mood + performance.

**MVP:** "Practice night template" that protects bedtime + suggests what to cut first ("minimum viable sleep mode").

## Nonprofit Concepts

### SF "Hoops Mental Toolkit" (Free Resource + Workshops)
**What:** Free site/app with short tools:
- Confidence reset
- Pre-game routine
- Post-game reflection
- Dealing with benching
- Injury comeback

**Approach:** Partner with school teams / rec centers, run 30-min workshops, publish toolkit for free.

**Impact Metrics:** # players served, workshop attendance, pre/post survey on confidence + stress.

---

### "Mentor Hoops" (Peer Mentorship Network)
**What:** Pair varsity players with younger hoopers for weekly check-ins (goal-setting + mindset + school balance).

**Approach:** Community mentorship, leadership, service hours.

**Impact Metrics:** # matches, retention, mentee goal completion.

## Skill Development Framework

### Step 1: Identify Weaknesses

**Methods:**
- Reflect
- Film review with coach
- Training feedback
- Playing with others

**Skill Categories:**

| Category | Sub-skills |
|----------|------------|
| **Decision Making** | Passing |
| **Ball Handling** | Full court pressure, ball manipulation |
| **Shooting** | Catch & shoot (L/R, off actions), off the dribble |
| **Finishing** | L/R hand layups, floaters |
| **Defense** | Help defense, closeouts |
| **Athleticism** | Speed, lateral quickness, vertical |

### Step 2: Improve Skills

**Principles:**
- Define a metric
- Build consistency
- Think beyond an app for solutions

**Example – Shooting:**
- High reps, mostly catch & shoot
- Use machine if available (auto-tracks)
- Form shots for mechanics
- **Metric:** Track makes / total shots weekly in one dedicated workout
