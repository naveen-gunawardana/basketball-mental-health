# Example: CloudFire Problem Interview (from Running Lean)

## Context
Ash Maurya interviewing bloggers about photo/video sharing pain points.

## Set the Stage
> "I'm researching how bloggers share photos and videos with readers. Not selling anything today—just learning."

## Demographics Collected
- Blog platform: WordPress
- Posts per week: 3-4
- Uses photos/video: Yes, in most posts
- Technical comfort: Moderate

## Problems Tested

| Problem | Rank | Acknowledged? |
|---------|------|---------------|
| Uploading large media files is slow | 1 | ✓ Yes |
| Media makes pages load slowly for readers | 3 | ✓ Yes |
| Managing/organizing media library is painful | 2 | ✓ Yes |

## Worldview Exploration

**Current solution**: "I resize everything in Photoshop before uploading. Takes forever."

**Frustration**: "I waste 30 min per post just on images. Should be writing."

**Frequency**: "Every single post. Multiple times a week."

**Cost**: "~2-3 hours/week. Plus I know my pages are still slow."

## Key Insights
- Problem #1 was universally acknowledged
- Existing alternatives (manual resizing) are painful but tolerated
- Time cost was felt more than technical complexity
- Referred 2 other bloggers for interviews
